1. Intro                                                     *vim-dpaste-intro*

Save selected text to dpaste service.

2. Requirements                                       *vim-dpaste-requirements*

curl - for http request

3. Usage                                                     *vim-dpaste-usage*

    1. Use visual mode for select text.
    2. Call :Dpaste command for save paste to dpaste service.
    3. Get snippet url from bottom of the window.
